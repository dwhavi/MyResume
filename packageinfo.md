# 기능

## 채용공고 분석
- 텍스트 붙여넣기로 채용공고 입력
- AI가 키워드 자동 분석
- 정합성 점수 도출 (공고 vs 프로필 매칭 시각화)

## 이력서 생성
- 사용자 경력/스킬 입력
- 최적화된 이력서 자동 생성
- 한글 이력서 포맷 준수

## PDF 내보내기
- A4 크기 PDF 생성
- 한글 폰트 지원
- 사진 배치 및 레이아웃 최적화

## 사진 업로드
- 사진 업로드, 자동 크롭, 압축
